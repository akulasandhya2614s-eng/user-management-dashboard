# 👨‍💼 User Management Dashboard

A modern and responsive **User Management Dashboard** built with **React.js** that allows administrators to perform complete **CRUD (Create, Read, Update, Delete)** operations on users using the JSONPlaceholder REST API.

This project demonstrates React fundamentals including API integration, component-based architecture, state management with hooks, form validation, searching, filtering, sorting, and responsive UI design.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/akulasandhya2614s-eng/user-management-dashboard.git
```

Move into the project

```bash
cd user-management-dashboard
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
